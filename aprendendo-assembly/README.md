# Aprendendo Assembly

Meu primeiro repositório para estudar Assembly do zero.

## Objetivo

Aprender como o computador executa instruções em baixo nível, entendendo:

- registradores
- memória
- instruções básicas
- syscalls
- loops
- comparações
- organização de código

## Ferramentas sugeridas

- NASM
- Linux, WSL ou uma máquina virtual Linux
- Git e GitHub
- VS Code ou outro editor de texto

## Como compilar no Linux/WSL

```bash
nasm -f elf64 arquivo.asm -o arquivo.o
ld arquivo.o -o programa
./programa
```

## Ordem dos estudos

1. Hello World
2. Registradores
3. Soma simples
4. Loops
5. Comparações
6. Funções
7. Pilha / Stack
8. Pequenos projetos
